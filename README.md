# Rogue Survivor

A roguelike mobile game for iOS, built with Swift and Xcode. Fight through waves of enemies, collect power-ups, and survive as long as you can.

## Overview

Rogue Survivor is an independently developed iOS mobile game built entirely from scratch using Swift. Inspired by classic roguelike games, the player navigates a character through increasingly difficult enemy waves, collecting power-ups and managing resources to stay alive. All game logic, UI, and assets were designed and programmed by the developer.

## Gameplay Features

- **Wave-based survival** — face progressively harder enemy waves
- **Enemy AI** — enemies navigate toward and engage the player dynamically
- **Player movement** — responsive touch-based controls
- **Power-up system** — collect items to boost stats and abilities
- **Custom UI & icons** — all interface elements and app icons designed by the developer
- **Increasing difficulty** — game scales in challenge the longer you survive

## Getting Started

### Requirements
- macOS with Xcode installed (Xcode 14+ recommended)
- iOS Simulator or a physical iPhone/iPad

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/BenjaminTia/Rogue-Survivor.git
   ```
2. Open `Rogue Survivor.xcodeproj` in Xcode
3. Select a simulator or connected device
4. Press `Cmd + R` to build and run
